# Phonebook Project

Welcome to the Phonebook project! This simple console-based application allows users to manage contacts, including phone numbers, optional messages, and personal diary entries. The project is implemented in Python and provides a user-friendly interface for performing various operations on the phone book.

## Features

- **Add Contact:** Add a new contact with a name, phone number, optional message, and personal diary entry.

- **Search Contact:** Search for a contact by name and view their details, including phone number, message, and diary entry.

- **Remove Contact:** Remove a contact from the phone book.

- **Update Contact:** Update the phone number, message, or diary entry of an existing contact.

- **Delete All Contacts:** Remove all contacts from the phone book.

- **Display All Contacts:** View a list of all contacts with their details.
